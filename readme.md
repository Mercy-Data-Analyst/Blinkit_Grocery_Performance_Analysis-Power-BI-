# Blinkit Grocery Performance Analysis (Power BI Dashboard)

## Project Overview
This project analyzes**Blinkit Grocery Data** to evaluate outlet performance,item category sales, fat content distribution, and yearly establishment trend using **Power BI**.

The dashboard provides interactive insights into sales distribution across outlet types, item types, and fat content categories to support data-driven decision-making

---

## Buisness Objectives

- How does outlet size and location impact performance?
- which outlet type contibutes the most revenue?
- Which item type generate the highest revenue?
- How does fat content influence total sales?
- Which trends exist across establishment year?
  
---

## Dataset Description

- Blinkit grocery dataset
- includes:
   - Item type
   - Fat content
   - Outlet size & location
   - Item identifier
   - Outlet type
   - Outlet establishment year
   - Outlet identifier
   - Sales
   - Rating

---

## Data Cleaning & Preparation
- Standardized inconsistencies
- Ensured proper data types
- Replaced null values
- Verified duplicates
- Created DAX measures for KPIs

---

## Key Metrics Created (DAX)
- Total Sales: $1.2M
- Average Sales: $141
- Number of Items: 16
- Average Rating: 4.0

---

## Key Insights

1. Location Insight
- Tier 3 outlet generate the highest sales($472k)
- Indicates strong demand in developing urban areas.

2. Outlet Type Insight
- Supermarket Type 1 dominates revenue ($787k)
- Grocery stores contribute significantly less

3. Item Performance
- Fruits & Vegetables and Snack foods are top-performing categories
- Diary and Frozen Foods show moderate contribution.

4. Fat Content Analysis
- Low Fat items generate higher overall revenue than Regular.
- Suggests consumer shift toward healthier options

5. Trend Insights
- Sales Peaked around 2018
- Slight decline post-2019
